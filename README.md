# Sentence Mining Helper

Browser-Erweiterung für Sentence Mining mit Firebase Cloud-Sync.

## Installation für Entwickler

### 1. Voraussetzungen
- Vivaldi/Chrome Browser
- Firebase Projekt (kostenlos)

### 2. Setup

```bash
# Repository klonen
git clone https://github.com/DEIN_USERNAME/sentence-mining-helper.git
cd sentence-mining-helper

# Firebase Konfiguration einrichten
cp firebase-config.template.js firebase-config.js
